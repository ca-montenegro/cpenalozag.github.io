Colmines

IoT application aimed at preventing health issues in mines by monitoring different variables and activating control systems as necessary. The system is able to handle emergency notifications from 2500 microcontrollers in 1 second with 0% error and a response time under 30 milliseconds. The system was built using architectural tactics to satisfy requirements on quality attributes:

Performance:

  • Introduce concurrency: Thread object pool

  • Multiple copies of computation

  • Resource scheduling

  • Managing event rate

Security:

  • User authentication

  • User authorization

  • Data confidentiality and integrity: Encryption and hashing

Modifiability:

  • Reduce coupling and increase cohesion

Usability:

  • Pagination

  • Maintain a model of the system

Availability:

  • Ping and monitoring

  • Active redundancy
